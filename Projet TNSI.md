Projet TNSI

**Thème imposé :** 

Jeu à deux joueurs sur un plateau, ou avec des cartes, des objets …

L’idée est d’avoir un espace partagé entre les deux joueurs pour jouer 

Un des joueurs peut gagner ou alors les deux joueurs peuvent gagner ensemble

**Travail à faire :**

\-      Niveau 1 : 

Réaliser le code nécessaire pour que 2 joueurs s’affrontent.

Un jeu dans un terminal est accepté , sinon avec Pygame

Critères : Originalité , Simplicité , Jouabilité 

\-      Niveau 2 : 

Réaliser le code pour qu’un joueur affronte un ordinateur . 
 L’ordinateur joue aléatoirement sans intelligence mais en veillant à respecter les règles

\-      Niveau 3 : 

On mémorise dans une file ou une pile les étapes du jeu.
 On doit pouvoir revenir en arrière en se servant de cette mémorisation du jeu.

\-      Niveau 4 :

Réaliser une IA pour jouer …
